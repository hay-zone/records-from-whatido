## 🎨 设计特点

根据访问分析，这个博客有以下设计特点：

### 1. 框架
- **VitePress** - Vue 3 + Vite 驱动的静态站点生成器

### 2. 视觉设计
- **主题配色**：El-Blue 主题
- **模式切换**：支持深色/浅色模式
- **布局**：三栏布局（左侧导航 + 中间内容 + 右侧大纲）

### 3. 核心功能

#### 🔍 搜索功能
- 全站搜索
- 覆盖 270+ 篇文章

#### 📢 公告弹窗
- 支持弹窗通知
- 展示微信/QQ 二维码

#### 🔥 热文推荐
- 精选 9 篇热门内容
- 首页展示

#### 💬 评论系统
- GitHub 评论集成

#### 📑 内容组织
- 按技术栈分类（力扣、Java、SpringBoot等）
- 左侧分类导航支持折叠

### 4. 页面元素
- Logo：简洁 SVG 设计
- 头像展示
- 格言："为学应尽毕生力，攀高须贵少年时"
