## 🖼️ 如何自定义背景和头像

### 方法 1: 使用本地图片

#### 1. 准备图片

将你的图片放到 `docs/public/` 目录：

```
docs/
└── public/
    ├── avatar.jpg        # 你的头像
    └── bg-image.jpg      # 背景图片
```

#### 2. 修改配置

编辑 `docs/.vitepress/config.js`:

```javascript
blog: {
  motto: '在代码与文字之间，拾取那些闪光的瞬间',

  // 使用本地头像（相对于 public 目录）
  avatar: '/records-from-whatido/avatar.jpg',

  // 使用本地背景（相对于 public 目录）
  bgImage: '/records-from-whatido/bg-image.jpg',

  slogan: '好好生活，慢慢相遇',
  showAuthor: true
}
```

### 方法 2: 使用外部图片 URL

直接使用图片链接：

```javascript
blog: {
  motto: '在代码与文字之间，拾取那些闪光的瞬间',

  // 使用外部 URL
  avatar: 'https://你的图片链接.com/avatar.jpg',
  bgImage: 'https://你的图片链接.com/bg-image.jpg',

  slogan: '好好生活，慢慢相遇',
  showAuthor: true
}
```

### 推荐的图片规格

**头像:**
- 推荐尺寸: 200x200 px 或更大的正方形
- 格式: JPG, PNG, WebP
- 文件大小: < 100KB

**背景图:**
- 推荐尺寸: 1920x1080 px 或更大
- 格式: JPG, PNG, WebP
- 文件大小: < 500KB
- 风格: 淡雅、低对比度（确保文字可读）

### SlovinG 风格的背景建议

根据图片，SlovinG 使用的是淡雅的水墨画风格背景。你可以：

1. **使用类似风格的图片**
   - 淡蓝色调
   - 水墨画风格
   - 低对比度
   - 柔和的色彩过渡

2. **推荐资源**
   - Unsplash: https://unsplash.com/s/photos/watercolor
   - Pixabay: https://pixabay.com/zh/images/search/水彩/
   - 搜索关键词: "watercolor background", "水墨画", "淡雅背景"

## 📝 博客文章 Frontmatter 配置

要让文章显示在博客首页，需要添加以下 frontmatter：

```markdown
---
title: 文章标题
date: 2026-01-13
tags:
  - 标签1
  - 标签2
categories:
  - 分类名
description: 文章摘要描述
cover: https://封面图片链接.jpg
sticky: 1           # 置顶（数字越大越靠前）
recommend: 1        # 推荐（会出现在精选文章列表）
---

# 文章标题

文章内容...
```

### 字段说明

- **title**: 文章标题（必填）
- **date**: 发布日期（必填，格式: YYYY-MM-DD）
- **tags**: 标签数组（可选）
- **categories**: 分类数组（可选）
- **description**: 文章摘要（可选，显示在卡片上）
- **cover**: 封面图片 URL（可选，显示在文章卡片）
- **sticky**: 置顶权重（可选，数字越大越靠前）
- **recommend**: 推荐权重（可选，会出现在精选文章列表）
