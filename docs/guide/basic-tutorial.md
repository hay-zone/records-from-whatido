# 基础教程

本教程将带您深入了解项目的核心功能和使用技巧，帮助您掌握日常开发中最常用的特性。

## Markdown 基础

VitePress 使用 Markdown 作为文档编写语言。Markdown 是一种轻量级标记语言，语法简单易学。您可以使用标题、列表、链接、图片等基本元素来组织内容。

### 标题层级

使用 `#` 符号定义标题，一个 `#` 表示一级标题，两个 `##` 表示二级标题，以此类推。建议保持标题层级的逻辑性，不要跳级使用。

### 文本格式

支持**粗体**、*斜体*、~~删除线~~等文本格式。您还可以使用行内代码 `code` 来标记代码片段。

## 代码块

VitePress 提供了强大的代码高亮功能，支持几乎所有主流编程语言：

```javascript
function hello() {
  console.log('Hello, VitePress!')
}
```

代码块支持行号显示、行高亮、代码组等高级功能，让您的技术文档更加专业。

## 自定义容器

使用自定义容器可以创建提示、警告、危险等特殊样式的内容块：

::: tip 提示
这是一个提示信息
:::

::: warning 警告
这是一个警告信息
:::

::: danger 危险
这是一个危险警告
:::

## 链接与导航

在文档中添加内部链接和外部链接非常简单。内部链接使用相对路径，VitePress 会自动处理路由。外部链接会在新标签页中打开。

## 图片资源

您可以在文档中插入图片，支持相对路径和绝对路径。建议将图片资源统一放在 `public` 目录下管理，这样可以更好地组织项目结构。

## 表格

Markdown 表格语法让您能够快速创建数据表格：

| 特性 | 说明 |
| --- | --- |
| 响应式 | 自动适配不同屏幕尺寸 |
| 主题化 | 支持自定义主题样式 |

## Front Matter

每个 Markdown 文件可以在开头使用 YAML 格式的 Front Matter 来定义页面元数据，如标题、描述、布局等信息。这些元数据会影响页面的渲染效果和 SEO 优化。

## 组件使用

VitePress 允许您在 Markdown 中直接使用 Vue 组件，这为创建交互式文档提供了无限可能。您可以导入自定义组件，或使用内置的特殊组件。

通过掌握这些基础知识，您已经可以创建出结构清晰、内容丰富的文档了。继续学习进阶指南，探索更多高级功能吧！
