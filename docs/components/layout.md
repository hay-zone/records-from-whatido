# 布局组件

布局组件提供了灵活的页面结构方案，帮助您快速搭建响应式的页面布局。本文档详细介绍各种布局组件的使用方法。

## 栅格系统

基于 24 栅格系统的布局方案，提供了强大的响应式布局能力：

```vue
<Row>
  <Col span="12">左侧列</Col>
  <Col span="12">右侧列</Col>
</Row>
```

通过设置不同的 `span` 值，可以灵活控制列的宽度。栅格系统会自动处理列之间的间距和对齐。

## 响应式布局

栅格系统支持响应式断点，可以为不同屏幕尺寸设置不同的布局：

```vue
<Col :xs="24" :sm="12" :md="8" :lg="6">响应式列</Col>
```

- **xs** - 超小屏幕（<576px）
- **sm** - 小屏幕（≥576px）
- **md** - 中等屏幕（≥768px）
- **lg** - 大屏幕（≥992px）
- **xl** - 超大屏幕（≥1200px）

## 容器组件

Container 组件提供了居中布局容器，限制内容的最大宽度，让页面在大屏幕上保持合适的阅读宽度：

```vue
<Container>
  <Header>页头</Header>
  <Main>主要内容</Main>
  <Footer>页脚</Footer>
</Container>
```

## 间距控制

通过 `gutter` 属性可以设置栅格列之间的间距：

```vue
<Row :gutter="16">
  <Col span="8">列1</Col>
  <Col span="8">列2</Col>
  <Col span="8">列3</Col>
</Row>
```

间距值以像素为单位，会自动在列之间平均分配。

## 对齐方式

行组件支持水平和垂直方向的对齐控制：

### 水平对齐

- **start** - 左对齐
- **center** - 居中对齐
- **end** - 右对齐
- **space-around** - 分散对齐（两端留空）
- **space-between** - 两端对齐（两端不留空）

### 垂直对齐

- **top** - 顶部对齐
- **middle** - 居中对齐
- **bottom** - 底部对齐

## 列偏移

通过 `offset` 属性可以让列向右偏移指定的栅格数：

```vue
<Col span="8" offset="4">偏移4格的列</Col>
```

这在需要留白或特殊布局时非常有用。

## 列排序

使用 `push` 和 `pull` 属性可以改变列的显示顺序，而不改变 DOM 结构：

```vue
<Col span="12" push="12">视觉上靠右</Col>
<Col span="12" pull="12">视觉上靠左</Col>
```

## Flex 布局

布局组件基于 Flexbox 实现，提供了现代化的布局能力。您可以使用 `flex` 属性进行更精细的控制：

```vue
<Row type="flex">
  <Col flex="1">弹性列1</Col>
  <Col flex="2">弹性列2（2倍宽度）</Col>
</Row>
```

## 嵌套布局

行和列可以任意嵌套，构建复杂的页面结构：

```vue
<Row>
  <Col span="16">
    <Row>
      <Col span="12">嵌套列1</Col>
      <Col span="12">嵌套列2</Col>
    </Row>
  </Col>
  <Col span="8">侧边栏</Col>
</Row>
```

## 实用技巧

合理使用布局组件可以大幅提升开发效率。建议在开发前先规划好页面的整体布局结构，选择合适的栅格划分方案。对于复杂布局，可以将其拆分成多个小的布局单元，分别实现后再组合。

布局组件是构建现代响应式网页的基石，掌握它们的使用方法，能让您的页面在各种设备上都有出色的表现。
