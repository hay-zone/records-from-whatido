# 更新日志

本页面记录了项目的所有重要更新和变更。我们遵循语义化版本规范，每个版本都包含新功能、改进和 bug 修复。

## v2.1.0 (2024-01-15)

### 新功能

- 添加了暗黑模式支持，用户可以根据系统设置或手动切换主题
- 实现了全文搜索功能，支持实时搜索和关键词高亮
- 新增了代码复制按钮，方便用户复制代码示例
- 支持了多语言切换，目前支持中文和英文

### 改进

- 优化了首页加载速度，减少了 30% 的初始加载时间
- 改进了移动端的阅读体验，调整了字体大小和行距
- 更新了代码高亮主题，支持更多编程语言
- 优化了侧边栏的滚动性能

### Bug 修复

- 修复了在 Safari 浏览器中图片加载失败的问题
- 修复了导航栏在小屏幕设备上显示异常的问题
- 修复了某些 Markdown 语法解析不正确的问题
- 修复了搜索功能在中文输入下的兼容性问题

## 版本规范

我们采用语义化版本号 (Semantic Versioning)：

- **主版本号 (Major)** - 不兼容的 API 修改
- **次版本号 (Minor)** - 向下兼容的功能性新增
- **修订号 (Patch)** - 向下兼容的问题修正

## 升级指南

### 从 v1.x 升级到 v2.x

由于 v2.0.0 采用了新的配置格式，升级时需要：

1. 更新 `package.json` 中的依赖版本
2. 运行 `npm install` 安装新版本
3. 迁移配置文件到新格式（参考迁移指南）
4. 检查并测试所有功能

### 从 v0.x 升级到 v1.x

v1.0.0 是首个稳定版本，建议所有用户升级：

1. 备份现有文档
2. 更新依赖并重新安装
3. 按照新的文档结构组织内容
4. 测试所有页面和功能
