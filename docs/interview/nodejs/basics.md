# Node.js 基础

## 1. 什么是 Node.js

Node.js 是基于 Chrome V8 引擎的 JavaScript 运行时环境。

### 特点
- 事件驱动
- 非阻塞 I/O
- 单线程(但可以创建子进程)

## 2. 核心模块

### fs (文件系统)
文件读写操作。

### http / https
创建 HTTP 服务器。

### path
路径处理。

### events
事件处理。

## 3. npm

Node Package Manager,JavaScript 包管理工具。

### package.json
项目配置文件。

### 常用命令
- `npm install`
- `npm run`
- `npm publish`
