# 机器学习基础

## 1. 什么是机器学习

机器学习是人工智能的一个分支,让计算机从数据中学习并做出预测。

### 类型
- 监督学习
- 无监督学习
- 强化学习

## 2. 常见算法

### 监督学习
- 线性回归
- 逻辑回归
- 决策树
- 随机森林
- 支持向量机(SVM)
- 神经网络

### 无监督学习
- K-means 聚类
- 层次聚类
- PCA(主成分分析)

## 3. 深度学习

### 神经网络
模拟人脑神经元的网络结构。

### CNN(卷积神经网络)
主要用于图像识别。

### RNN(循环神经网络)
主要用于序列数据处理。

### Transformer
现代 NLP 的基础架构。

## 4. 常用框架

- TensorFlow
- PyTorch
- Keras
- Scikit-learn
