# 计算机视觉

## 1. 基础概念

计算机视觉是让计算机能够"看懂"图像和视频的技术。

### 主要任务
- 图像分类
- 目标检测
- 语义分割
- 实例分割
- 姿态估计

## 2. 经典模型

### 图像分类
- ResNet
- VGG
- Inception
- EfficientNet

### 目标检测
- YOLO
- Faster R-CNN
- SSD
- RetinaNet

### 语义分割
- U-Net
- DeepLab
- FCN

## 3. 图像预处理

- 归一化
- 数据增强
- 裁剪和缩放
- 色彩调整

## 4. 应用场景

- 人脸识别
- 自动驾驶
- 医疗影像分析
- 图像搜索
- OCR(文字识别)
- 视频监控
