# records-from-whatido

基于 VitePress 构建的个人在线资料库文档站点。

## 项目简介

这是一个使用 VitePress 构建的现代化文档网站，包含完整的导航结构和丰富的内容。

## 功能特性

- 🚀 基于 Vite 构建，极速开发体验
- 📝 Markdown 编写，简单易用
- 🎨 响应式设计，支持多端访问
- 🔍 内置搜索功能
- 📦 五大主题模块，涵盖全面内容

## 内容结构

- **指南** - 快速开始、基础教程、进阶指南
- **组件** - 按钮、表单、布局组件
- **API 参考** - 核心 API、工具 API、配置 API
- **示例** - 基础示例、进阶示例、最佳实践
- **关于** - 项目介绍、团队成员、更新日志

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览构建产物
npm run docs:preview
```
